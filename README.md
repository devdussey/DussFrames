# DussFrames
Modern, neat and lightweight frames for WoW.

[![Duss-Frames-Showcase.png](https://i.postimg.cc/MH1MCt8j/Duss-Frames-Showcase.png)](https://postimg.cc/FksHcyz9)

## Currently:
- Basic Player and Target Frame with resource bar.
- Currently, your class colour is default and only option.
- Size is default

## Commands:
- /df lock - Lock all frames
- /df unlock - Unlock all frames
- /df lock player - Lock player frame only
- /df lock target - Lock target frame only
- /df unlock player - Unlock player frame only
- /df unlock target - Unlock target frame only
- /df status - Show lock status of both frames
- /df opacity 0-100 - Set frame opacity (0-100%)
